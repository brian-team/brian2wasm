Brian2Wasm: examples

This page demonstrates the use of Brian2Wasm, a package to create WebAssembly code from Brian simulations.

Brunel & Hakim (1999)

All of the following examples reproduce a simulation from the paper “Fast Global Oscillations in Networks of Integrate-and-Fire Neurons with Low Firing Rates” (Brunel & Hakim, 1999).

  1. Basic simulation: simulation with automatically generated HTML page and default options
  2. Parameter change: changing simulation parameters
  3. Realtime feedback: displaying results continuously during a simulation run
  4. Custom style: customized HTML page and a plot with chart.js
